## 第五件事：`run_bash`函数 → 安全执行终端命令
**对应代码**：def run_bash(command: str) -> str
**核心作用**：AI调用终端命令时，做安全防护+执行命令+返回结果
**执行步骤**：
1. 检查命令是否危险（删系统、重启等），危险则直接拦截
2. 尝试执行bash命令，设置超时120秒、捕获输出
3. 捕获超时、命令不存在等异常，返回错误信息
4. 合并命令的正常输出和错误输出，超长则截断后返回

```python

def run_bash(command: str) -> str:
    dangerous = ["rm -rf /", "sudo", "shutdown", "reboot", "> /dev/"]
    if any(item in command for item in dangerous):
        return "Error: Dangerous command blocked"
    try:
        result = subprocess.run(
            command,
            shell=True,
            cwd=os.getcwd(),
            capture_output=True,
            text=True,
            timeout=120,
        )
    except subprocess.TimeoutExpired:
        return "Error: Timeout (120s)"
    except (FileNotFoundError, OSError) as e:
        return f"Error: {e}"
    output = (result.stdout + result.stderr).strip()
    return output[:50000] if output else "(no output)"

```

## 第六件事：`extract_text`函数 → 提取AI的纯文本回复
**对应代码**：def extract_text(content) -> str
**核心作用**：过滤AI响应里的工具调用信息，只保留人类能看懂的文字
**执行步骤**：
1. 判断输入内容是否为列表，不是则返回空
2. 遍历AI响应的所有内容块
3. 只提取纯文字部分，忽略工具调用块
4. 拼接所有文字，去除空格后返回

```python

def extract_text(content) -> str:
    if not isinstance(content, list):
        return ""
    texts = []
    for block in content:
        text = getattr(block, "text", None)
        if text:
            texts.append(text)
    return "\n".join(texts).strip()

```

## 第七件事：`execute_tool_calls`函数 → 执行AI要求的工具调用
**对应代码**：def execute_tool_calls(response_content) -> list[dict]
**核心作用**：解析AI的指令，自动运行bash命令，并把结果返回给AI
**执行步骤**：
1. 遍历AI的响应内容，找到「工具调用」指令
2. 提取AI要执行的bash命令，彩色打印给用户看
3. 调用`run_bash`安全执行命令
4. 把命令结果整理成AI能识别的格式，返回给程序

```python

def execute_tool_calls(response_content) -> list[dict]:
    results = []
    for block in response_content:
        if block.type != "tool_use":
            continue
        command = block.input["command"]
        print(f"\033[33m$ {command}\033[0m")
        output = run_bash(command)
        print(output[:200])
        results.append({
            "type": "tool_result",
            "tool_use_id": block.id,
            "content": output,
        })
    return results

```

## 第八件事：`run_one_turn`函数 → 执行AI智能体的单轮交互
**对应代码**：def run_one_turn(state: LoopState) -> bool
**核心作用**：完成一轮「AI思考→调用工具→记录结果」的操作
**执行步骤**：
1. 调用Claude API，发送对话历史，获取AI响应
2. 把AI的回复保存到状态记录本
3. 判断AI是否需要调用工具：不需要则结束任务
4. 需要工具则调用`execute_tool_calls`执行命令
5. 把命令结果保存到对话历史，轮次+1，标记继续循环


```python

def run_one_turn(state: LoopState) -> bool:
    response = client.messages.create(
        model=MODEL,
        system=SYSTEM,
        messages=state.messages,
        tools=TOOLS,
        max_tokens=8000,
    )
    state.messages.append({"role": "assistant", "content": response.content})
    if response.stop_reason != "tool_use":
        state.transition_reason = None
        return False
    results = execute_tool_calls(response.content)
    if not results:
        state.transition_reason = None
        return False
    state.messages.append({"role": "user", "content": results})
    state.turn_count += 1
    state.transition_reason = "tool_result"
    return True

```
