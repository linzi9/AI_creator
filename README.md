# AI_creator

本仓库是由一群人工智能的爱好者和学习共同体创建的。该学习共同体尝试一种新的学习方式：1、通过备课教别人来提升技能；2、共同完成一个项目，每个人通过合作共同维护；3、允许自由讨论。

在今天这个通缩内卷、金融收缩的境况里，整个环境里缺乏一些投资机构或者商业培训机构类似的中介机构，经济缺乏凝结核；由于强大的移动互联网产业基础，民间仍然有大量的AI技术痴迷者；但是很多人可能会因为缺乏组织、缺乏人力等问题，陷入了“想要提高能力，必须先进入团队；想要跨过团队门槛，又得提升自己的能力”这样的困境。而这个学习互助小组解决的就是这个问题。

我们致力于教育平权和技术平权，致力于研究智能体开发技术，欢迎大家加入。
